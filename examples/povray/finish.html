From the POVray manual:

The finish properties of a surface can greatly affect its appearance. How does light reflect? What happens in shadows? What kind of highlights are visible. To answer these questions you need a finish.

The following code is written into the finish section of each object in the .pov file (the actual values may vary depending on the object type): 

finish {
  ambient 0.10
  diffuse 0.70 brilliance 1.00
  specular 0.00 roughness 0.01
}

For a complete treatment of the finish statement and all its keywords, please consult the POVray manual. Here only the effect of changing some of the above values is shown.
